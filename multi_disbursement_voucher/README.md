# Multi Disbursement Voucher for Odoo 17

## الوصف
هذه الوحدة تتيح إنشاء سندات صرف متعددة البنود في Odoo 17، حيث يمكن صرف مبالغ مختلفة لحسابات أو شركاء مختلفين ضمن سند واحد، مع إنشاء القيود المحاسبية المقابلة تلقائياً.

## المميزات
- رقم تسلسلي تلقائي لكل سند.
- إمكانية إضافة أسطر صرف متعددة.
- دعم الحسابات التحليلية والشركاء في كل سطر.
- إنشاء قيد محاسبي واحد (Journal Entry) عند التأكيد (Post).
- يظهر السند في قائمة المحاسبة -> العملاء (Accounting -> Customers).

## طريقة التثبيت
1. قم بتحميل الملف المضغوط `multi_disbursement_voucher.zip`.
2. فك الضغط عن الملف وانقل مجلد `multi_disbursement_voucher` إلى مسار الوحدات المخصصة لديك (Addons Path).
3. فعل وضع المطور (Developer Mode) في Odoo.
4. اذهب إلى قائمة **التطبيقات (Apps)**.
5. انقر على **تحديث قائمة التطبيقات (Update Apps List)**.
6. ابحث عن `Multi Disbursement Voucher` وقم بتثبيته.

## المتطلبات التقنية
- Odoo 17 (Community or Enterprise).
- وحدة المحاسبة (account) مثبتة مسبقاً.
